Resumo das opções principais:
git add .: Adiciona todos os arquivos do diretório atual (modificados e novos).
git add nome_do_arquivo: Adiciona um arquivo espec�fico.
git add -u: Adiciona apenas arquivos já rastreados que foram modificados.
git add -A ou git add --all: Adiciona todos os arquivos (novos e modificados).
git add -p: Adiciona partes de arquivos (modo interativo).
git add -i: Modo interativo para escolher alterações específicas.
